1. ข้อมูลที่เราเก็บรวบรวม

แอปพลิเคชัน CheckDarn เก็บรวบรวมข้อมูลดังต่อไปนี้:

  • ข้อมูลตำแหน่ง: เพื่อแสดงกล้องจราจรในบริเวณใกล้เคียง
  • ข้อมูลบัญชีผู้ใช้: ชื่อ อีเมล จาก Google Sign-In
  • รายงานจากผู้ใช้: ข้อมูลรายงานกล้องจราจรที่ผู้ใช้ส่ง
  • ข้อมูลการใช้งาน: เพื่อปรับปรุงประสิทธิภาพแอป

2. วัตถุประสงค์ในการใช้ข้อมูล

เราใช้ข้อมูลของคุณเพื่อ:

  • ให้บริการแสดงตำแหน่งกล้องจราจร
  • ให้ผู้ใช้สามารถรายงานและตรวจสอบกล้องได้
  • ปรับปรุงคุณภาพและประสิทธิภาพของแอป
  • ป้องกันการใช้งานที่ผิดวัตถุประสงค์

3. การแบ่งปันข้อมูล

เราไม่แบ่งปันข้อมูลส่วนบุคคลของคุณให้กับบุคคลที่สาม ยกเว้น:

  • เมื่อได้รับความยินยอมจากคุณ
  • เพื่อปฏิบัติตามกฎหมายหรือคำสั่งศาล
  • ในกรณีฉุกเฉินเพื่อความปลอดภัยสาธารณะ

4. การรักษาความปลอดภัยข้อมูล

เราใช้มาตรการรักษาความปลอดภัยที่เหมาะสม เช่น:

  • การเข้ารหัสข้อมูลระหว่างการส่งผ่าน
  • การใช้ Firebase Security Rules
  • การจำกัดการเข้าถึงข้อมูล

5. สิทธิของผู้ใช้

คุณมีสิทธิ:

  • ขอเข้าถึงข้อมูลส่วนบุคคลของคุณ
  • ขอแก้ไขข้อมูลที่ไม่ถูกต้อง
  • ขอลบข้อมูลของคุณ
  • ขอถอนความยินยอม

6. การลบบัญชีและข้อมูล

หากคุณต้องการลบบัญชีและข้อมูลทั้งหมดของคุณจากระบบ CheckDarn:

  • ส่งอีเมลไปที่: checkdarn.app@gmail.com
  • ระบุหัวข้อ: "ขอลบบัญชี CheckDarn"
